# 🛒 Amazon Store Annual Report Dashboard (2022)

## 📌 Project Overview
This project is an interactive **Amazon Store Annual Report Dashboard (2022)** built in **Power BI** to analyze sales performance, customer behavior, and order trends.

The dashboard provides insights into:
- Sales Performance 📈
- Order Analysis 📦
- Customer Demographics 👥
- Sales by State 🌍
- Sales Channels Performance 🛍️
- Product Category Analysis 🏷️

## 📊 Dashboard Highlights

### KPI Metrics
- **Total Sales:** 719,147  
- **Total Orders:** 1,045  
- **Average Order Value:** 688.18  
- **Total Customers:** 1,045  

## 📈 Key Insights

### Order vs Sales Analysis
- Monthly sales and orders trend
- Peak sales observed in **October and December**

### Gender Analysis
- **Women:** 63%
- **Men:** 37%

### Top Sales States
- Maharashtra
- Karnataka
- Uttar Pradesh
- Telangana
- Tamil Nadu

### Sales Channel Contribution
- Amazon – 37%
- Myntra – 23%
- Flipkart – 22%
- Ajio – 6%
- Nalli – 5%
- Meesho – 4%
- Others – 3%

### Orders by Age & Gender
- Adult Women contribute highest orders (**35.69%**)
- Adult Men contribute **21.44%**
